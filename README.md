# tgpostmaker

Telegram-бот для подготовки постов в канал: выбор канала, добавление inline-кнопок, немедленная публикация и отложенный автопостинг.

## Возможности

- Доступ только для Telegram ID из `ADMIN_IDS`.
- Выбор канала через Telegram-кнопку с запросом админ-прав для бота.
- Проверка, что бот действительно может публиковать в выбранный канал.
- Черновик поста создаётся из одного сообщения: текст, фото, видео, документ или другое одиночное медиа.
- Кнопки под постом задаются текстом:

```text
!r Купить - t.me/shop | !g Подробнее - https://example.com
!b Написать - tg://resolve?domain=username
```

- Поддержка цветов кнопок:
  - `!r` — красная кнопка (`danger`)
  - `!g` — зелёная кнопка (`success`)
  - `!b` — синяя кнопка (`primary`)
- Планирование:
  - `сегодня 18:00`
  - `завтра 09:30`
  - `через 2 часа`
  - `25.06 14:00`

## Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Скопируйте `.env.example` в `.env`.
3. Заполните переменные:

```env
BOT_TOKEN=123456789:replace-with-your-token
ADMIN_IDS=123456789,987654321
TZ=Europe/Minsk
DB_PATH=/data/bot.db
SCHEDULER_INTERVAL_SECONDS=5
```

`ADMIN_IDS` — это Telegram ID пользователей, которым разрешено пользоваться ботом.

## Запуск через Docker Compose

```bash
docker compose up -d --build
```

Посмотреть логи:

```bash
docker compose logs -f bot
```

Остановить:

```bash
docker compose down
```

SQLite-база хранится в папке `data`, которая монтируется в контейнер как `/data`.

## Как пользоваться

1. Напишите боту `/start`.
2. Выберите канал через кнопку `Выбрать канал`.
3. Telegram предложит добавить бота администратором канала. Нужны права на публикацию сообщений.
4. Нажмите `Автопост`.
5. Пришлите будущий пост одним сообщением.
6. Добавьте кнопки, запланируйте публикацию или нажмите `Опубликовать`.

Если время не выбрано, `Опубликовать` отправит пост сразу. Если время выбрано, `Опубликовать` поставит пост в очередь.
